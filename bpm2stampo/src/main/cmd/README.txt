bpm2rdf-converter utility help

Usage
1. Drag and drop
	Drag the file that you want to convert on the bpm2rdf-converter.bat 
	file. A console window will open automatically. When the conversion is 
	done the windows will close. output file - the output file will be in 
	the directory of the drag&dropped *.bpm file. 

	
2. Command line
	Single argument
	
		> bpm2rdf-converter.bat processmodel.bpm
	
	The output will be processmodel.rdf
	
	Specify the name of the output:
	
		> bpm2rdf-converter.bat input.bpm output.rdf
	
	Note
		make sure the batch file "bpm2rdf-converter.bat" is on the path.
